Font-feature-settings

ソースを見る スペックを見る

font-feature-settings ミックスインは、 OpenType フォントに含まれることのある高度なタイポグラフィー機能を使う際に便利です。

{% highlight scss %} // 書体とフォントファイルが合字を含んでいれば使用する @include font-feature-settings("liga"); // 数字にプロポーショナルフォントを使い、自動カーニングは行わない @include font-feature-settings("pnum", "kern" false); {% endhighlight %}
