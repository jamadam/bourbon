Font-feature-settings

ソースを見る スペックを見る

font-feature-settings ミックスインは、 OpenType フォントに含まれることのある高度なタイポグラフィー機能を使う際に便利です。

{% highlight scss %} // Use ligatures if the typeface and font file include them @include font-feature-settings("liga"); // Use proportional numbers, but not automatic kerning @include font-feature-settings("pnum", "kern" false); {% endhighlight %}
