Perspective

ソースを見る スペックを見る

CSS の perspective プロパティは、3D 配置された要素に遠近感を与えるため、z=0 平面とユーザとの間の距離を決めます。

CSS の perspective-origin プロパティは、ユーザの見ている位置を決めます。これは perspective プロパティによって消失点として使われます。

{% highlight scss %} @include perspective(300px); @include perspective-origin(30% 30%); {% endhighlight %}
