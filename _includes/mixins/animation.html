animation ミックスインは、個々のプロパティの異なったトランジションを一つのスタイルルールに記述できるよう、カンマ区切りの値をサポートします。リスト中のそれぞれの値は、他方のプロパティ群の同じ位置の値に対応します。

{% highlight scss %} box:hover { // Animation の短縮記法は、CSS3 の animation 短縮記法と同様に動作する。 @include animation(scale 1.0s ease-in, slide 2.0s ease); // 上の例は、独立の細かいミックスインを使用した場合と同様の結果になる。 @include animation-name(scale, slide); @include animation-duration(2s); @include animation-timing-function(ease); @include animation-iteration-count(infinite); } {% endhighlight %}
