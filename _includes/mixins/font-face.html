Font-face

ソースを見る スペックを見る

シンプルな @font-face ミックスインは、カスタムフォントの拡張を簡単にします。使用例のとおり。

このミックスインは Rails アセットパイプラインと合わせた用途のために、オプショナルな引数 $asset-pipeline: true をとります。

{% highlight scss %} @include font-face(SourceSansPro, '/fonts/Source_Sans_Pro/SourceSansPro-Regular'); @include font-face(SourceSansPro, '/fonts/Source_Sans_Pro/SourceSansPro-Bold', bold); @include font-face(SourceSansPro, '/fonts/Source_Sans_Pro/SourceSansPro-Italic', normal, italic); // Rails asset-pipeline - place fonts in app/assets/fonts/ @include font-face(SourceSansPro, 'Source_Sans_Pro/SourceSansPro-Regular', normal, $asset-pipeline: true); {% endhighlight %}
