Border Width

ソースを見る

border-width ミックスインは、null を含む最大4つの幅を受け取り、directional-property ミックスインを使って、それぞれの方角にマッピングします。

{% highlight scss %} @include border-width(1em 20px null 100%); {% endhighlight %}

CSS出力

{% highlight scss %} border-top-width: 1em; border-right-width: 20px; border-left-width: 100%; {% endhighlight %}
