Position

ソースを見る

ページ内の要素の位置を設定する省略記法です。

第1引数は省略可能で、 relative がデフォルト値です。第2引数はCSSの標準的な省略記法を踏襲したスペース区切りの値リストです。

注: null 値は無視されます。次の例では、right と bottom の位置のためのセレクタは生成されないことを意味します。一方で、topの位置は 0px になります。

こう書く代わりに

{% highlight scss %} position: relative; top: 0px; left: 100px; {% endhighlight %}

こう書けます

{% highlight scss %} @include position(relative, 0px null null 100px); {% endhighlight %}
