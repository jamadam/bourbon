Directional-property

ソースを見る

方角のプロパティのための短縮記法を提供するヘルパーミックスインです。接頭辞と接尾辞、最大4つの値の配列を受け取り、それぞれ top, right, bottom, left にマッピングします。null を接尾辞引数に渡すことで、これを無視させることができます。null を方角の値に渡すことで、これを無視させることができます。

このミックスインは主に、他のミックスインからヘルパーとして使用されます。 border-color 、 border-style 、 border-width 、 margin 、 padding を参照してください。

{% highlight scss %} @include directional-property(border, width, 10px null 4px 3px); {% endhighlight %}

CSS出力

{% highlight scss %} border-top-width: 10px; border-bottom-width: 4px; border-left-width: 3px; {% endhighlight %}
