Retina Image

ソースを見る

retina-image ミックスインは、Retina用の background-image と非Retina用の background-image を生成するヘルパーです。Retina用の background Image は HIDPI メディアクエリーに出力されます。このミックスインは、 ファイル名の接尾辞のデフォルトとして、 _2x.png を使用します。

$filename が、画像へのパスを解決します(例: "../../home-icon")。Railsで asset-pipeline を使用するには、該当の引数に true を与えます。

{% highlight scss %} @retina-image($filename, $background-size, $extension*, $retina-filename*, $retina-suffix*, $asset-pipeline*) {% endhighlight %}

* = 省略可

引数のデフォルト

  • $extension: png
  • $retina-filename: null
  • $retina-suffix: _2x
  • $asset-pipeline: false
{% highlight scss %} span { @include retina-image(home-icon, 32px 20px); } {% endhighlight %}

CSS出力

{% highlight css %} span { background-image: url(home-icon.png); } @media only screen and (-webkit-min-device-pixel-ratio: 1.3), only screen and (min--moz-device-pixel-ratio: 1.3), only screen and (-o-min-device-pixel-ratio: 1.3 / 1), only screen and (min-resolution: 125dpi), only screen and (min-resolution: 1.3dppx) { span { background-image: url(home-icon_2x.png); background-size: 32px 20px; } } {% endhighlight %}
