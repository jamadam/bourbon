Hide-text

ソースを見る

hide-text は、画像の置き換えのミックスインです。これは HTML5-Boilerplate image-replacement をベースとしています。

この画像置き換えの手法は、 text-indent の手法の代替です。これには パフォーマンとその他の利点 があります。

この動作のためには、height 宣言が必須です。(REQUIRED)

{% highlight scss %} div { @include hide-text; background-image: url(logo.png); height: 40px; } {% endhighlight %}
