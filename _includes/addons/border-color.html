Border-color

ソースを見る

border-color ミックスインは、null を含む最大4つの値を受け取り、directional-property ミックスインを使って、それぞれの方角にマッピングします。

{% highlight scss %} @include border-color(red green null blue); {% endhighlight %}

CSS出力

{% highlight scss %} border-top-color: red; border-right-color: green; border-left-color: blue; {% endhighlight %}
