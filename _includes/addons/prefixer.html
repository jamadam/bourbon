Prefixer

ソースを見る

prefixer は、ベンダープレフィックス付きの宣言を生成します。prefixer はプレフィックスとして webkit moz ms o spec を受け付けます。

{% highlight scss %} @mixin box-sizing($box) { @include prefixer(box-sizing, $box, webkit moz spec); } {% endhighlight %}

CSS出力

{% highlight scss %} -webkit-box-sizing: $box; -moz-box-sizing: $box; box-sizing: $box; {% endhighlight %}
