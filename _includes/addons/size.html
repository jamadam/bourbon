Size

ソースを見る

width と height を単一のステートメントで設定します。auto や inherit、単位なしの数値、 fill や max-content 、 min-content 、 fit-content などの内在的（ intrinsic ）キーワードを含む全ての単位を受け付けます。このミックスインは、 calc() CSS 関数との併用も可能です。

{% highlight scss %} @include size(2em); // width: 2em; height: 2em; @include size(10em auto); // width: 10em; height: auto; @include size(min-content 9rem); // width: min-content; height: 9rem; @include size(20px calc(100% - 80px)); // width: 20px; height: calc(100% - 80px); {% endhighlight %}
